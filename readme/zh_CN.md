# Taiwan Stock
[繁體中文](/README.md) [简体中文](/readme/zh_CN.md) [English](/readme/English.md)

## 介绍
这是一款可以针对台湾证券交易所的股票信息进行呈现，将您想看到的股票显示出来。

## 系统要求
- Python3.6 或更高的版本!

## 安装与执行
```
# install requirments
cd ./taiwan-stock/
pip install -r requirements.txt
python main.py
```
